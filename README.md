# Log Level at Antigravity

1. Open the Antigravity command palette mode (Press `Ctrl+Shift+P` or `F1`).
2. Type `Developer: Set Log Level...` and hit Enter.
3. Choose `Extension`, then find and select `rust-analyzer`.
4. Set it to `Trace` or `Debug`.
